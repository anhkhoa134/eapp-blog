Seed post images
================

All `.jpg` files in this folder were generated locally with ComfyUI at
`http://127.0.0.1:8188` for domain-app seed data.

Generation setup:

- Diffusion model: `flux-2-klein-4b-Q4_K_M.gguf`
- Text encoder: `qwen_3_4b_fp4_flux2.safetensors`
- VAE: `flux2-vae.safetensors`
- Output format: 1200x675 JPEG blog hero images

The images are intended for demo/seed content only and contain no third-party
product photos copied from external websites.
