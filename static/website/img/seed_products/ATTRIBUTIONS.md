# Seed Product Image Attributions

These images are local seed/demo assets for product data.

All `.jpg` files in this folder were generated locally with ComfyUI at `http://127.0.0.1:8188` using the Flux2 Klein GGUF workflow:

- diffusion model: `flux-2-klein-4b-Q4_K_M.gguf`
- text encoder: `qwen_3_4b_fp4_flux2.safetensors`
- VAE: `flux2-vae.safetensors`

The images are product-shot style assets generated specifically for this demo seed catalog.
